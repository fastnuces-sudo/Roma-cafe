## 1. Project Overview

**Aroma Café** is a simple multi-page business website created using pure **HTML**, **CSS**, and **JavaScript**.  
It is designed for an imaginary café to showcase:

- Home page introduction
- Menu / products
- About / story and team
- Contact details with a validated contact form

The project avoids heavy frameworks so that the code is easy to understand and explain in an ICT viva.

---

## 2. Features

- **4 Main Pages**
  - `index.html` – Home
  - `menu.html` – Menu (services/products)
  - `about.html` – About
  - `contact.html` – Contact

- **Semantic HTML5 Structure**
  - Uses `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`.

- **Custom CSS Styling**
  - Flexbox and Grid layouts
  - Responsive design for mobile and tablet
  - Consistent color theme and typography

- **JavaScript Interactivity**
  - Contact form validation (name, email, subject, message, phone)
  - Menu filtering by category (coffee, tea, desserts, snacks)
  - “Back to top” button
  - Dynamic current year in footer

- **Clean, Commented Code**
  - Easy to read and explain in viva or presentation.

---

## 3. Technologies Used

- **HTML5** – Structure and content
- **CSS3** – Styling, layout, responsiveness
- **JavaScript (ES5/ES6)** – Client-side interactivity and validation

No external CSS frameworks (like Bootstrap) are used to keep the project simple and original.
